# String

## Overview

Strings are a primitive datatype that hold data that can be represented in text form.

### Creation

Create a String with double quotes:

```misti
val name = "John"
```

### Interpolation

String interpolation is performed with curly braces.

```misti
val fullName = "{name} Doe"  // John Doe
```
