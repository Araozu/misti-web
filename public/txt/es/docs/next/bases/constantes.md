# Constantes y Variables

## Constantes

> En uso

```
const nombre = "Pedro"
```

Anidado:

```
const nombreCompleto =
    const nombre = "Juan"
    const apellido = "Perez"

    nombre + apellido

```

## Variables

> En uso

```
let edad = 20

edad = 21  // OK
```

No se puede declarar una variable y dejarla sin valor (por ahora)

```
let edad   //! Error. Se esperaba el operador de asignacion '='

// ...
```
